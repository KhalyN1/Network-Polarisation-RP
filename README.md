# Detecting network polarisation using cross-community social tie

#### This repository was made as part of the 2025-2026 Edition of the CSE3000 Research Project course at the Delft University of Technology.

#### We study the correlation between cross-community social ties and polarisation in synthetic networks, generated using the Stochastic Block model and LFR Benchmark model.

#### The framework used for measuring polarisation is inspired by the work of Mepham et al. (2025).
